## üéØ Specific Use Case Examples

### Example 1: Marketing Manager Planning Campaign

**User Query**: "I need to plan a social media campaign for a new product launch targeting millennials"

**How Infinity Assistant Helps**:
1. **Research**: Searches knowledge base for campaign strategies
2. **Gap**: No millennial-specific campaign data found
3. **Web Search**: Searches latest millennial marketing trends
4. **Synthesis**: Combines knowledge base patterns + web research
5. **Recommendations**: Provides campaign strategy with:
   - Platform recommendations (TikTok, Instagram)
   - Content types (video, stories, reels)
   - Timing strategies
